# 航空CBT培训平台

基于HTML的飞机交互式系统培训课件管理系统

## 功能特点
### 学员端
- 登录/个人中心
- 课程列表/章节导航
- 课件播放/学习计时
- 测试功能/成绩查看
- 学习记录/统计

### 管理端
- 仪表盘/数据统计
- 学员管理（增删改查/批量导入导出）
- 课程管理（CBT/ACPC/PBN三类课程）
- 课件管理（ZIP包上传/HTML课件）
- 课程分配/学习记录
- 报表导出

## 技术架构
- 前端：纯HTML + CSS + JavaScript，无需后端
- 存储：LocalStorage，数据保存在浏览器本地
- 部署：纯静态文件，支持Vercel/GitHub Pages/Nginx等任意静态服务器

## 部署方式
### Vercel部署
1. 上传到GitHub仓库
2. 连接Vercel，自动部署
3. 无需后端配置，直接访问

### 本地运行
1. 下载压缩包解压
2. 用浏览器打开index.html
3. 或使用静态服务器：`npx serve .`

## 课程分类
- CBT：飞机系统交互式培训课件
- ACPC：副驾驶预备课程
- PBN：性能导航培训

## 课件格式
上传ZIP压缩包，必须包含index.html作为入口文件，支持HTML/JS/CSS/图片/视频等资源。

## 学习记录
- 自动记录学习开始/结束时间
- 统计有效学习时长
- 章节完成状态跟踪
- 支持CSV导出
